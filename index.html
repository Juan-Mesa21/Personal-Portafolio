01. Fibonacci Sequence

Calculate the first 'n' terms of the Fibonacci sequence using my custom Python algorithm translated for the web.

[0, 1, 1, 2, 3]
Source: main.py PYTHON 
def get_fibonacci_sequence(n):
    if n <= 0: return []
    elif n == 1: return [0]
    
    seq = [0, 1]
    while len(seq) < n:
        next_val = seq[-1] + seq[-2]
        seq.append(next_val)
    return seq
